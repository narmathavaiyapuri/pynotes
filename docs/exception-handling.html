Exception Handling

Tue 21 April 2026 
# Basic try-except
try:
    num = int("abc")
except:
    print("Error occurred")

Error occurred

# Specific exception
try:
    a = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")

Cannot divide by zero

# finally block
try:
    x = int("100")
    print(x)
except:
    print("Error")
finally:
    print("Done execution")

100
Done execution
