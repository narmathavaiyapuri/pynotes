```python
#Adding element
fruits = ["apple", "banana"]

print("Updated list:", fruits)
```

    Updated list: ['apple', 'banana', 'orange']
    


```python
## Example 3: Loop through list
marks = [80, 90, 75]

for m in marks:
    print("Mark:", m)
```

    Mark: 80
    Mark: 90
    Mark: 75
