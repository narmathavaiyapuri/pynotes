```python
#Basic tuple
numbers = (1, 2, 3)

print("Tuple:", numbers)
print("First element:", numbers[0])
```

    Tuple: (1, 2, 3)
    First element: 1
    


```python
#Tuple with different data types
data = (10, "hello", 3.5)

print("Tuple:", data)
print("Second element:", data[1])
```

    Tuple: (10, 'hello', 3.5)
    Second element: hello
    


```python
#Loop through tuple
marks = (80, 90, 75)

for m in marks:
    print("Mark:", m)
```

    Mark: 80
    Mark: 90
    Mark: 75
