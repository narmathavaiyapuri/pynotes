```python
# For loop
for i in range(5):
    print("Number:", i)
```

    Number: 0
    Number: 1
    Number: 2
    Number: 3
    Number: 4
    


```python
# While loop
i = 0

while i < 5:
    print("Value:", i)
    i += 1
```

    Value: 0
    Value: 1
    Value: 2
    Value: 3
    Value: 4
    


```python
# Loop through list
fruits = ["apple", "banana", "orange"]

for fruit in fruits:
    print("Fruit:", fruit)
```

    Fruit: apple
    Fruit: banana
    Fruit: orange
